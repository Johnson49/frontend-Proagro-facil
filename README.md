## Começando

### Clone o projeto

```
git clone https://github.com/Johnson49/frontend-Proagro-facil
```

#### Adentre no diretório 

```shell
cd frontend-Proagro-facil
```

####  Instale as dependências

```node
npm install 
```

##  Inicie o servidor 

```
 npm run dev
```
